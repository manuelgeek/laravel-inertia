## About Laravel Inertia

 This is a sample Laravel project to illustrate Creating SPA with Inertia JS
 
 ## Set up 
 
 Fork and Clone the project  then;
 
 ```bash
composer install

cp .env.example .env

php artisan key:generate

php artisan migrate 

npm run dev

php artisan serve

```

## Features 
* Register
* Login
* Update Avatar
* CRUD
* Pagination
